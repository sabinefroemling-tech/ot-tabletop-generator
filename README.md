# OT-Tabletop-Übungsgenerator · OT Tabletop Exercise Generator

Erzeugt aus wenigen Angaben ein vollständiges Moderations-Drehbuch für eine OT-Tabletop-Übung: Ablaufplan nach dem Sieben-Phasen-Aufbau, Einspielungen (Eintrittsvektor × Zielsystem × Branche), Containment-Dilemma nach No-Touch-Logik, optional BSI-Meldepflicht- und Medien-Komplikation, Beobachterbogen, Debrief-Leitfaden mit Lessons-Learned-Raster (NIST CSF v2) und spoilerfreier Einladung.

Zweisprachig Deutsch/Englisch — Umschalter oben rechts, direkt verlinkbar über `?lang=en`.

**Live:** https://sabinefroemling-tech.github.io/ot-tabletop-generator/

Companion zum Fachbuch **„ISMS für die Industrie — Der Praxisratgeber"** (Springer Vieweg, 2026), Kap. 8.4 (Tabletop-Übungen), 8.1.1 (OT-Vorfall-Triage), 9 (No-Touch-Register), 8.3.1 (Frühwarnung §32 BSIG), 17.4 (Tabletop als Audit-Stufe 4).

Alle Eingaben werden clientseitig im Browser verarbeitet; eine Übertragung an einen Auswertungsserver ist nicht vorgesehen.
